# depviz

## Synopsis

Quick and dirty module dependency visualization with Graphviz.

## Description

## Requirements

- GraphViz

## Installation

```bash
npm install depviz -g
```

## Notes

Alternatively you might want to use:

```
$ git clone express
$ cd express && npm install && npm ls
```

## Tests

### Running the tests

```bash
$ npm test
```

### Test coverage

```bash
$ npm run-script coverage
```
